# Relembrando comandos git/github

## Configurações 
Adciona o nome do desenvolvedor:
````bash
git config --global user.name "Abner Oliveira"
````
Adciona o e-mail do desenvolvedor:
````bash
git config --global user.email abnernrt1@gmail.com
````

## Comandos Git

serve para salvar tudo
````bash
git add .
````
init serve para inicar o repositorio
````bash
init
````
git status serve para verificar se deu certo
````bash
git status
````
git commit -m "first commit" salva todos os comandos usados
````bash
git commit -m "first commit"
````
